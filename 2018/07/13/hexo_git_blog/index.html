Centos7.2 hexo + Github 搭建个人博客

1.准备工作

1.1安装gcc-g++

yum install gcc-c++

1.2安装nodejs

wget https://nodejs.org/dist/v8.1.2/node-v8.1.2.tar.gz
tar -zxvf node-v8.1.2.tar.gz
cd node-v8.1.2
mkdir -p /opt/node
./configure –prefix=/opt/node/
make && make install
ln -s /opt/node/lib/node_modules/npm/bin/npm-cli.js /usr/bin/npm
ln -s /opt/node/bin/node /usr/bin/node

1.3配置GitHub

git config –global user.name “xiaoshen344”
git config –global user.email “xiaoshen344@126.com“
将ssh的公钥拷贝至GitHub
可使用ssh -T git@github.com命令验证是否配置成功

1.4安装hexo

npm install hexo-cli -g
ln -s /opt/node/lib/node_modules/hexo-cli/bin/hexo /usr/bin/hexo
cd /opt/node/
hexo init blog
cd blog/
npm install
hexo server # 启动hexo
npm install hexo-deployer-git –save

1.5_config.yml中配置deploy

deploy:
: type: git
repo: git@github.com:xiaoshen344/xiaoshen344.github.io.git
branch: master

1.6将博客部署至GitHub

hexo clean #清除缓存文件 (db.json) 和已生成的静态文件 (public)
在某些情况（尤其是更换主题后），如果发现您对站点的更改无论如何也不生效，您可能需要运行该命令。
hexo generate #生成静态文件
hexo g
hexo server #启动服务器。默认情况下，访问网址为 http://localhost:4000/
hexo s
hexo deploy #部署到 github
hexo d
